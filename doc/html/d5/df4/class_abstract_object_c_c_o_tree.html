Detailed Description

Abstract tree with vessel structures represented by objects. Its slower than AbstractStructuredCCOTree although has more functionality and is more customizable.

virtual int AbstractObjectCCOTree::testVessel ( point  xNew,
AbstractVascularElement *  parent,
AbstractDomain *  domain,
vector< AbstractVascularElement * >  neighbors,
double  dlim,
point *  xBif,
double *  cost 
)
pure virtual

For a given spatial point xNew test its connection with parent vessel. It must evaluate if the restrictions of geometry and symmetry are satisfied and also if it do not intersects with other vessel of this tree. It returns in cost the functional value with the inclusion of such vessel.

Parameters
xNewDistal point for the new vessel to test.
parentParent vessel to test the xNew point connection.
domainTree domain.
neighborsClose neighbors used for intersection test.
dlimNot used in the current implementation.
xBifProximal point for the new vessel that present the lower function cost.
costFunctional cost variation for the best bifurcation position.
Returns
If the connection of the tree with xNew is possible. If not cost is INFINITY.
