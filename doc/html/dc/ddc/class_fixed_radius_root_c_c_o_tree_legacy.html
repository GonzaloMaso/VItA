Detailed Description

Models a single CCO tree which contains the vessel's data and connectivity. The implementation corresponds to the algorithm 3 of Queiroz Ph.D. Thesis. Current implementation is endowed with OpenMP parallelism. Specifications:

  • Fixed domain;
  • Given flow Q;
  • Given pressure gradient;
  • Given N terminals;
  • Constant/variable symmetry per tree level;
  • Constant/variable Murray coefficient per tree level;
  • Constant/variable viscosity per tree level.

For a given spatial point xNew test its connection with parent vessel. It must evaluate if the restrictions of geometry and symmetry are satisfied and also if it do not intersects with other vessel of this tree. It returns in cost the functional value with the inclusion of such vessel.

Parameters
xNewDistal point for the new vessel to test.
parentParent vessel to test the xNew point connection.
domainTree domain.
neighborsClose neighbors used for intersection test.
dlimNot used in the current implementation.
xBifProximal point for the new vessel that present the lower function cost.
costFunctional cost variation for the best bifurcation position.
Returns
If the connection of the tree with xNew is possible. If not cost is INFINITY.

Tests N_BIF_TRIES * (N_BIF_TRIES+1)/2 - 3 possible positions for the bifurcation site in the triangle delineated by the triplet (xNew, parent->xProx, parent->xDist). It returns the bifurcation site xBif which corresponds to the minimum cost in terms of volumetric area of the tree. If all bifurcations are invalid, cost will be DBL_MAX.
