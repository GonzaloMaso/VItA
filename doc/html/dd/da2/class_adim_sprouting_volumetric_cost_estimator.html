Detailed Description

Cost estimator that considers diffusion and vessel-wall degradation associated to sprouting angiogenesis and also weighs the variation of the tree volume. Wall degradation is assumed linear w.r.t. vessel radius (under the assumption that vessel thickness is linear with vessel radius) and media diffusion of biosignals is assumed to follow Fick's 2nd law.
